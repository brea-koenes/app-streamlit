# Store Stockout Classifier

A machine learning application that identifies store-level stockouts through Uber Eats customer reviews.

## About

This classifier detects stockout signals in customer feedback that traditional inventory systems miss—enabling proactive detection, faster store response, and revenue recovery at scale.

## How to Run Locally

1. **Install dependencies**:
    ```bash
    pip3 install -r requirements.txt
    ```

2. **Start the app**:
    ```bash
    streamlit run main.py
    ```
    
    Or if `streamlit` isn't in your PATH:
    ```bash
    python3 -m streamlit run main.py
    ```
